LeadHunter SaaS

v5.5 · Live

SaaS-система для поиска, обработки и экспорта лидов из Telegram. Проект закрывает путь от подключения аккаунта до сбора контактов, фильтрации, сегментации и выгрузки данных для дальнейшей работы.

Ключевые решения:

  • ✓ Multi-Account: пул аккаунтов с переключением при ограничениях Telegram
  • ✓ Anti-Ban 5.0: паузы, batch-обработка, обработка FloodWait
  • ✓ SaaS-изоляция: пользователи работают только со своими источниками и лидами
  • ✓ Экспорт: выгрузка лидов в CSV/XLSX для передачи в работу

Что показывает проект:

Работа с Telegram API, асинхронной обработкой, пользовательскими кабинетами, хранением данных, антиспам-логикой, экспортом и построением продукта вокруг автоматизации.

AI-панель для управления Telegram-контентом

ZodiacOps · MVP

Продуктовая система для автоматизации контент-процесса в Telegram: от идеи и генерации черновика до проверки качества, ручного ревью, планирования публикации и контроля очереди. Главная ценность проекта не в названии, а в управляемом pipeline, который помогает выпускать контент регулярно и безопасно.

Что умеет система:

  • ✓ Генерация черновиков: создание постов по теме, платформе и контексту канала
  • ✓ Review workflow: статусы draft, needs review, approved, rejected, scheduled, published, failed
  • ✓ Quality Check: оценка качества, флаги проблем, ручное approve/reject без автопубликации
  • ✓ Publishing Queue: очередь постинга, сортировка по времени, retry для ошибок и защита от дублей

Инженерная часть:

Backend на FastAPI с моделями контента, каналов и очереди публикаций. Web-панель на React/Vite. Очереди и фоновые задачи через Redis/Celery. Интеграция с Telegram Bot API, форматирование постов под Telegram, диагностика ошибок и оформление задач/баг-репортов в Jira.
